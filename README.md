## Description

Backend of the Overcraft application: User creation, build order creation, and in-game display.

## Technologies Used

| Function   | Name        |
|------------|-------------|
| Framework  | NestJs      |
| Database   | PostgreSQL  |
| ORM        | Prisma      |
| Deployment | Docker      |

## Local Development Setup

Make a copy of the ".env.template" file and fill in all the environment variables.
